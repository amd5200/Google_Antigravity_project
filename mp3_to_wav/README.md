# 文字轉語音 (WAV) 轉換器 (Text to Speech to WAV Converter)

> **⚠️ 注意：這個程式是專門為 RadioMaster 遙控器設計支援（專用）的語音轉換工具。**

這個專案提供了一個具備圖形化介面 (GUI) 的輕量級工具，能讓你輕鬆將輸入的文字轉換為語音，並儲存為 WAV 格式的音訊檔。

## 功能特色
- 支援任意文字內容轉為語音 (Text-to-Speech)。
- 輕巧直覺的圖形化介面，操作簡單。
- 自動處理轉檔，直接輸出符合以下規格的 WAV 音訊檔案：
  - **單聲道 (Mono)**
  - **32kHz 採樣率**
  - **16-bit PCM 格式**
  - **自動正規化音量** (避免聲音過小或破音)

## 如何編譯或執行這個程式？

你可以選擇使用打包好的獨立執行檔，或是直接執行 Python 原始碼。

### 方式一：直接使用桌面應用程式 (.exe)

如果你只想直接使用軟體，不需要配置任何環境：
1. 進入開發目錄下打包好的 `dist` 資料夾。
2. 雙擊執行 `mp3_to_wav.exe` 即可直接打開軟體！
3. *(註：如果你還沒有打包過，可以雙擊專案目錄下的 `build.bat` 檔案，系統會自動幫你將 Python 程式與套件打包，並生成 `dist/mp3_to_wav.exe`。)*

### 方式二：透過 Python 環境執行

如果在開發階段或想透過 Python 執行，請按照以下步驟：

1. **安裝依賴套件**
   打開終端機（命令提示字元或 PowerShell），進入這個專案所在的資料夾，輸入以下指令：
   ```bash
   pip install -r requirements.txt
   ```

2. **啟動主程式**
   套件安裝完成後，輸入這行指令啟動程式介面：
   ```bash
   python mp3_to_wav.py
   ```

## 軟體操作說明

當你啟動軟體，看到「**文字轉語音(WAV)轉換器**」的介面後：
1. **輸入文字**：在最上方的「輸入文字」多行文字框中，輸入或貼上你想要轉換為語音的文字。
2. **設定儲存位置**：點擊「儲存 WAV 路徑」右側的「**瀏覽...**」按鈕，選擇你要儲存 WAV 音訊檔的位置與檔名。
3. **開始轉換**：確認文字與路徑都輸入後，按下最下方綠色的「**開始轉換**」按鈕。
4. **轉換完成**：轉換期間按鈕會顯示「轉換中...」並暫時無法點擊。等待畫面彈出「文字成功轉換為語音 WAV 檔！」的提示對話框，就代表語音檔已經成功生成在你的指定位置囉！
