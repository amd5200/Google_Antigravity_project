# 音訊處理與轉換工具 (文字/MP3轉WAV)

> **⚠️ 注意：這個程式是專門為 RadioMaster 遙控器設計支援（專用）的語音轉換工具。**

這個專案提供了一個具備圖形化介面 (GUI) 的輕量級工具，能讓你輕鬆將「輸入的文字轉換為語音」，或是將「既有的 MP3 音訊檔導入轉換」，並儲存為符合特定規格的 WAV 音訊檔。

## 功能特色
- 支援任意文字內容轉為語音 (Text-to-Speech)。
- 支援導入 MP3 (或其他常見音訊) 檔案並自動轉檔為 WAV。
- 輕巧直覺的圖形化介面分為多個頁籤，操作簡單。
- 自動處理轉檔，直接輸出符合以下規格的 WAV 音訊檔案：
  - **單聲道 (Mono)**
  - **32kHz 採樣率**
  - **16-bit PCM 格式**
  - **自動正規化音量** (避免聲音過小或破音)

## 如何編譯或執行這個程式？

你可以選擇使用打包好的獨立執行檔，或是直接執行 Python 原始碼。

### 方式一：直接使用桌面應用程式 (.exe)

如果你只想直接使用軟體，不需要配置任何環境：
1. 進入開發目錄下打包好的 `dist` 資料夾。
2. 雙擊執行 `mp3_to_wav.exe` 即可直接打開軟體！
3. *(註：如果你還沒有打包過，可以雙擊專案目錄下的 `build.bat` 檔案，系統會自動幫你將 Python 程式與套件打包，並生成 `dist/mp3_to_wav.exe`。)*

### 方式二：透過 Python 環境執行

如果在開發階段或想透過 Python 執行，請按照以下步驟：

1. **安裝依賴套件**
   打開終端機（命令提示字元或 PowerShell），進入這個專案所在的資料夾，輸入以下指令：
   ```bash
   pip install -r requirements.txt
   ```

2. **啟動主程式**
   套件安裝完成後，輸入這行指令啟動程式介面：
   ```bash
   python mp3_to_wav.py
   ```

## 軟體操作說明

當你啟動軟體，看到介面後，可以根據需求選擇上方頁籤：

### 📝 模式一：文字轉語音
1. **輸入文字**：在「輸入文字」多行文字框中，輸入或貼上你想要轉換的文字。
2. **設定儲存位置**：點擊「儲存 WAV 路徑」右側的「**瀏覽...**」按鈕，選擇你要儲存的位置與檔名。
3. **開始轉換**：按下最下方的「**開始轉換**」按鈕，成功後即可生成語音檔。

### 🎵 模式二：音訊檔轉 WAV
1. **選擇來源檔案**：點擊右側的「**瀏覽...**」按鈕選擇現有的 MP3 音訊檔案（系統會自動幫您填寫預設的輸出路徑）。
2. **確認儲存位置**：需要的話，可以在「儲存 WAV 路徑」重新設定或修改輸出位置。
3. **開始轉換**：按下藍色的「**開始轉換**」按鈕，就會自動幫您轉換好符合 RadioMaster 規格的音訊檔。
